## Obsidian Git 插件

[Obsidian Git](https://github.com/Vinzent03/obsidian-git) 是 [[Obsidian]] 的社区插件。

### 功能

- 自动定时备份到 [[GitHub]]
- 启动 Obsidian 时自动 pull
- 保存笔记后自动 commit + push

### 安装

Obsidian → 设置 → 第三方插件 → 社区插件市场 → 搜索 "Obsidian Git" → 安装

搭配 [[Quartz]] 使用可实现写完即发布的体验。
