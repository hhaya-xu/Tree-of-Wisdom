## Quartz

[Quartz](https://quartz.jzhao.xyz) 是一个快速的静态网站生成器，专为 [[Obsidian]] 笔记设计。

### 核心特性

- **知识图谱**：自动生成交互式知识图谱
- **[[双向链接]]** 支持
- **全文搜索**
- **反向链接**：每篇笔记底部自动显示哪些页面链接了当前页面
- **标签系统**
- **RSS 订阅**

### 工作原理

```
本地 Obsidian 写 Markdown → git push → GitHub Actions 构建 → GitHub Pages 发布
```
