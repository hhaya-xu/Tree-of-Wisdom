## 2026-05-14 ~ 2026-05-23 · 基础设施搭建

### 决策

- 选型 Obsidian + Quartz v4 + GitHub Pages，全免费
- 仓库命名 Tree-of-Wisdom，不用 my-knowledge-base
- GitHub Actions 部署使用 `peaceiris/actions-gh-pages` 方案（后弃用）
- 最终采用 GitHub 原生 `actions/deploy-pages` + `actions/upload-pages-artifact`

### 踩坑

1. **GitHub Pages 部署失败（红×）**：原因是仓库无默认分支。解决：Settings → Branches 创建 main 分支
2. **环境保护规则拒绝部署**：`github-pages` environment 未配置。解决：Settings → Environments → github-pages → 移除分支限制
3. **Node 版本告警**：GitHub Actions 用 actions/upload-artifact@v4（Node 20，已弃用）。暂未修复，2026-06-02 前需升级到 v5+
4. **Obsidian Git 插件初次使用**：主公不熟悉 Git 概念。解决：配好 SSH 远程 + 10 分钟自动 push

- 仓库：`git@github.com:hhaya-xu/Tree-of-Wisdom.git`
