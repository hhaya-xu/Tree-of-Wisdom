## 新项目启动清单

**模板：**
1. `content/` 下新建项目文件夹
2. 写 `定题说明.md`（项目边界、人物、尺度）
3. 建 `原始材料/`、`策展文本/`、`术语库/` 子目录
4. 写第一篇策展文本
5. 壳自动识别新文件夹，生成导航入口

**日志条目：** 2026-05-24 · 架构 + 壳 · 多项目策略

## CSS 壳扩展模板

**模板：** 为新页面类型加样式时，在 `custom.scss` 末尾追加：

```scss
/* [页面类型] */
[data-slug*="路径匹配"] .center article {
  /* 特定规则 */
}
```

所有 Grid/列宽规则包在 `@media (min-width: 1200px)` 内。

**日志条目：** 2026-05-24 · 架构 + 壳 · CSS 壳方案

## 单主题站点锁定

**经验：** 如果站点只有一套视觉方案（如纯暗色），不要在 UI 上留切换开关。在根上锁死：

1. 移除切换按钮（UI 层）
2. `lightMode = darkMode`（配置层，HEX 值以兼容 OG 图片生成）
3. 强制脚本：`saved-theme="dark"` + `nav` 事件兜底（运行时层）

**适用场景：** Quartz、或其他有主题切换功能的 SSG。

**日志条目：** 2026-05-24 · 暗色锁定
