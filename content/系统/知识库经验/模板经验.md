# 模板经验

> 从项目日志中萃取的「可复用的结构模板」经验。

---

## 策展文本 frontmatter 模板

**模板：**
```yaml
---
title: "Day N · 日期"
date: YYYY-MM-DD
description: "一句话概述"
tags: [策展, 时间线]
---
```

**来源：** 2026-05-24 MD 规范。

---

## 术语条目 MD 模板

**模板：**
```markdown
---
title: 术语名
date: YYYY-MM-DD
tags: [术语]
aliases: [别名1, 别名2]
---

## 定义

## 出处

## 关联

- [[相关术语1]]
- [[相关术语2]]
```

**来源：** 入流亡所策展 术语库实践。

## 新项目启动清单

**模板：**
1. `content/` 下新建项目文件夹
2. 写 `定题说明.md`（项目边界、人物、尺度）
3. 建 `原始材料/`、`策展文本/`、`术语库/` 子目录
4. 写第一篇策展文本
5. 壳自动识别新文件夹，生成导航入口

**来源：** 2026-05-24 多项目策略 + 文件夹即模板。

---

## CSS 壳扩展模板

**模板：** 为新页面类型加样式时，在 `custom.scss` 末尾追加：

```scss
/* [页面类型] */
[data-slug*="路径匹配"] .center article {
  /* 特定规则 */
}
```

所有 Grid/列宽规则包在 `@media (min-width: 1200px)` 内。

**来源：** 2026-05-24 CSS 壳方案。
