## 二、技术架构

```
Obsidian（本地写作）
    │  Markdown + Wikilink + Frontmatter
    ▼
Git（版本管理 + 协作）
    │  git push → GitHub
    ▼
Quartz v4.5.2（静态站点生成）
    │  npx quartz build → HTML + CSS + JS
    ▼
GitHub Pages（免费托管）
    │  自动部署
    ▼
https://hhaya-xu.github.io/Tree-of-Wisdom/
```

### 关键机制

- **写作**：Obsidian 中写 `.md`，用 `[[双向链接]]` 建立关联
- **图谱**：Quartz 自动渲染知识图谱，节点 = 页面，边 = wikilink
- **部署**：`git push` 触发 GitHub Actions，自动 build + 部署
- **协作**：他人通过 GitHub PR 提交内容修改

## 四、CSS 架构（当前视觉实现）

### 样式文件分层

| 文件 | 职责 |
|------|------|
| `quartz.config.ts` | 全站字体（Fraunces 标题 / Geist 正文 / Geist Mono 代码）、色彩变量 |
| `quartz/styles/custom.scss` | 全局布局覆盖 + 视觉风格（~600 行） |
| `quartz/components/styles/topbar.scss` | 顶栏专属样式 |
| `quartz/components/styles/sidebar-zones.scss` | 右栏图谱/内容切换区 |
| Quartz 核心 scss | `base.scss`、`variables.scss` 等（不直接改，只覆盖） |

### 布局结构

```
┌──────────────────────────────────────────────┐
│  🌿 智慧树  │  入 流 亡 所 群 聊 录  │  🔍   │  ← header 90px
│  黑底 #09090a · 金字 oklch(78% 0.075 82)    │      sticky
├────────┬─────────────────────┬───────────────┤
│ 左栏    │  中栏（唯一可滚）     │  右栏          │
│ sticky  │  overflow-y: auto   │  sticky        │
│         │                     │               │
│ 日期/   │  正文内容             │  [内容区]      │
│ 阅读时长 │                     │  [图谱区]      │
│ ─────  │                     │               │
│ 目录树  │                     │               │
├────────┴─────────────────────┴───────────────┤
```

- Grid: `260px 1fr 320px` / rows `auto 1fr auto`
- 视口锁定：`html, body { height: 100vh; overflow: hidden }`
- 仅中栏滚动
