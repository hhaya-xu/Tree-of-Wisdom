# MD 样式映射表

> 在 Obsidian 中用以下标记书写，Quartz 自动渲染为对应样式。
> 字体统一由 `quartz.config.ts` 控制：标题 Fraunces / 正文 Geist / 代码 Geist Mono。

## 一、字体层级

| 标记 | HTML 标签 | 字体 | 字号 | 字重 |
|------|----------|------|------|------|
| `# 一级标题` | `<h1>` | Fraunces | 1.8rem | 500 |
| `## 二级标题` | `<h2>` | Fraunces | 1.35rem | 500 |
| `### 三级标题` | `<h3>` | Fraunces | 1.1rem | 500 |
| 正文段落 | `<p>` | Geist | 继承 | 400 |
| `**粗体**` | `<strong>` | Geist | 继承 | 600 |
| `*斜体*` | `<em>` | Geist | 继承 | 400 |
| `` `代码` `` | `<code>` | Geist Mono | 0.88rem | 400 |

## 二、链接

| 标记 | 效果 |
|------|------|
| `[[目标页面]]` | 内部链接，琥珀色高亮底色 |
| `[[目标页面\|显示文字]]` | 别名链接 |
| `[外部链接](url)` | 外部链接，琥珀色 |

## 三、块元素

| 标记 | 效果 |
|------|------|
| `> 引用` | 左侧琥珀边框 + 浅琥珀底色 |
| `---` | 分割线 |
| `- 无序列表` | 标准列表 |
| `1. 有序列表` | 编号列表 |
| `` ``` 代码块 ``` `` | 浅灰背景代码块 |

## 四、表格

| 特性 | 样式 |
|------|------|
| 表头 `<th>` | Fraunces 500，底部分隔线 |
| 单元格 `<td>` | Geist，浅底部分隔线 |

### 对话格式

| 格式 | CSS 类 | 说明 |
|------|--------|------|
| 时间戳 | `.msg-ts` | 灰色小字等宽 |
| 对话内容 | `.msg-content` | 正文色，行高 1.9 |
| 对话块 | `.dialogue-block` | 左侧细线缩进 |
| 精华摘录 | `.excerpt-line` | 斜体大字 |
