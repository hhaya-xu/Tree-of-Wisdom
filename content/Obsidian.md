## Obsidian

[Obsidian](https://obsidian.md) 是一款基于本地 Markdown 文件的知识库工具。

### 核心特性

- **[[双向链接]]**：用 `[[笔记名]]` 语法连接笔记，形成知识网络
- **知识图谱**：可视化笔记之间的关联关系
- **本地优先**：所有笔记都是纯 Markdown 文件，完全由你掌控
- **插件生态**：丰富的社区插件，如 [[Obsidian Git 插件]]

### 与 Quartz 的配合

Obsidian 负责写作体验，[[Quartz]] 负责将笔记发布为网站。
两者共享同一套 Markdown 文件，完美协作。
