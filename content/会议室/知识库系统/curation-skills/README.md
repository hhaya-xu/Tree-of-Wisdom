## ⑥ 的三层管线

```
项目日志（记） → 收拾桌子（炼） → 知识库经验（沉淀）
content/系统/    伯喈定期执行      content/系统/知识库经验/
项目日志.md                        ├── 分类经验.md
  纯事实                           ├── 索引经验.md
  不带评价                         └── 模板经验.md
```

- 项目日志：伯喈在每次重要决策后记录事实
- 收拾桌子：伯喈定期从日志中提炼经验，写入三个经验文件
- 知识库经验：沉淀为可复用的规则和模板
