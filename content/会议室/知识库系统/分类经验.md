## 2026-05-29 经验

### Quartz SCSS 编译顺序
组件 SCSS → custom.scss → joinStyles :root
后面覆盖前面，所以同一选择器只在 custom.scss 改才生效。

### 提交验证
每次 build + push 后必须 `git show --stat` 确认提交包含了预期文件。
Node.js writeFile + 后续 shell git add 可能出现 CRLF 导致文件未被纳入。

### 编码安全
SCSS 文件不要用 PowerShell Set-Content 写（破坏 UTF-8）→ 只能用 Node.js writeFile。
